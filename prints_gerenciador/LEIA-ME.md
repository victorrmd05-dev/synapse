# 📸 Prints do Gerenciador de Eventos (Meta)

Jogue aqui os screenshots do Gerenciador de Eventos pra eu analisar (dedup, EMQ,
eventos chegando, configurações do Pixel, etc.).

## Como usar
1. Tire o print (Windows: `Win + Shift + S` pra recortar, ou `PrtScn`).
2. Salve a imagem **dentro desta pasta** (`prints_gerenciador/`).
3. Formatos que eu leio: **PNG, JPG/JPEG**.
4. Me diga o nome do arquivo (ou só diga "subi o print") que eu abro e analiso.

## Dica de nome (opcional, ajuda a organizar)
`aba-testar-eventos.png`, `config-pixel.png`, `pageview-dedup.png`, etc.

> Esta pasta é só pra inspeção — fica de fora do git (ver `.gitignore`).

# Common Tools Skill — General-Purpose Utilities

This skill provides a set of general-purpose tools and utilities that can be accessed by any Alavanca AI agent. It aims to optimize common tasks such as web research, file manipulation, basic communication, and other functionalities not specific to a single department.

## TOOLS AND PROCEDURES

### 1. Research and Information Gathering

•Advanced Web Search: Use search tools to gather updated information, market trends, competitor data, or any other relevant public data.

•Content Extraction: Scripts to extract text from URLs, PDFs, or other online documents for analysis.

•Information Summarization: Tools to summarize long articles, reports, or documents, extracting key points.

### 2. Data and File Manipulation

•Format Conversion: Utilities to convert files between different formats (e.g., Markdown to PDF, CSV to JSON).

•Text Analysis: Basic tools for word counting, sentiment analysis, or entity identification in text blocks.

•File Management: Scripts to organize, rename, or move files within the agent's working environment.

### 3. Communication and Notifications

•Simple Report Generation: Tools to format data into basic reports in Markdown or CSV.

•Internal Notifications: Mechanisms to send simple notifications or alerts to other agents or to the  [CEO](https://paperclip.zedocarro.cloud/ALA/agents/ceo)  about task status or important events.

## AGENT MENTALITY

When using the Common Tools Skill, the agent should strive for efficiency and automation of repetitive tasks. The mindset is to leverage available resources to accelerate workflow and free up time for more complex and strategic activities.
